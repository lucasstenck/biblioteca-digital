# Biblioteca Digital

Aplicativo móvel para download de livros em PDF.

## Funcionalidades

- Cadastro e login de usuários
- Listagem de livros
- Download de livros em PDF
- Interface para Android e iOS

## Tecnologias

- React Native, Expo, TypeScript
- Node.js, Express, MySQL

## Instalação

1. Clonar o repositório
2. Instalar dependências: `npm run install-all`
3. Configurar banco MySQL
4. Iniciar: `npm start`

## Estrutura

- `backend/` - API Node.js
- `frontend/` - App React Native
- `database/` - Scripts de banco
